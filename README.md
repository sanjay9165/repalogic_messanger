# repalogic_messanger

A simple real-time chat application built with Flutter, Firebase, and Riverpod. Users can sign in with Google, create chat rooms, invite other users, and exchange real-time messages.

## Features
- Google Sign-In authentication
- User profiles stored in Firestore
- Create and join chat rooms
- Invite users to existing rooms
- Real-time messaging using Firestore streams
- Simple and clean UI built with Flutter

## How to run the application

- Run the app on a connected device or emulator:
   ```bash
   flutter run
   ```
   - **Android**: Make sure an Android emulator is running or a device is connected.

This project is intended as a sample messenger-style application demonstrating Firebase-backed authentication and chat using Flutter and Riverpod.
